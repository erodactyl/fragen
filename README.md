# Commands

## `fragen config`

Add the OpenAI API key to your config file. Required to use the `fragen` command.

```
USAGE
  $ fragen config

DESCRIPTION
  Configure OpenAI API key.
```

## `fragen`

Start a chat session with ChatGPT, required OpenAI API key to be configured.

```
USAGE
  $ fragen
```
